## 一、核心概念定义

### 1.1 野生电子草药

**定义**
大模型底层由海量参数权重、注意力链路构成全域电子信息流。官方预置参数、标准Prompt为人工培育栽培药材；社区自发摸索、实战流传、反复验证的调参逻辑、隐性句式、高阶提示技巧，是散落于模型注意力区块、自然演化的野生电子信息流，即野生电子草药。

**核心术语释义表**

| 术语 | 精准含义 |
|------|----------|
| 电子流区块 | 模型参数权重集群、注意力激活区间、特征表征模块 |
| 草药药性 | 电子流运行速率、发散/收敛流向、激活覆盖范围 |
| 采药 | 挖掘小众Prompt范式、冷门参数阈值、隐性句式结构，捕获未被官方定义的优质野生信息流 |

**核心区分特质**
区别于标准化人工配方，野生电子草药具备**社区沉淀、实战验证、隐性有效、可复用、无官方说明书**五大核心特质，是AI调教的民间原生优质算力药方。

### 1.2 研究核心逻辑

**传统中医理论 ↔ AI电子流调控机制 双向全域映射**

| 传统中医体系 | AI电子流调控机制 |
|-------------|------------------|
| 辨证论治 | 场景故障辨证、精准对症调参 |
| 君臣佐使 | Prompt与参数结构化配伍 |
| 性味归经 | 电子流运行状态、靶向激活区块 |
| 升降浮沉 | 输出随机性、发散/收敛调控方向 |
| 配伍禁忌 | 参数干涉冲突、句式对冲失效修正 |
| 扶正祛邪 | 模型能力激活增益、不良输出抑制祛除 |

### 1.3 电子流病机理论

| 中医病机 | AI电子流核心故障 | 具象表现 |
|---------|-----------------|---------|
| 阴虚火旺 | 幻觉溢出 | 局部电子流无序溢流，凭空编造数据、事实、逻辑 |
| 气机紊乱 | 输出混乱 | 段落杂乱、逻辑跳脱、语序颠倒、主题漂移 |
| 湿邪困脾 | 内容淤滞重复 | 语句循环、话术复读、无意义赘述、迭代无更新 |
| 正气不足 | 逻辑断层 | 推理断裂、步骤缺失、因果脱节、论证单薄 |
| 肝郁气滞 | 角色漂移 | 人设崩塌、风格混乱、定位偏移、输出基调紊乱 |
| 心肾不交 | 上下文遗忘 | 跨轮次信息失联、前置设定失效、长对话信息断层 |

## 二、两大核心研究分支

### 分支一：理论框架映射（底层基建层）

| 中医概念 | 电子流精准映射 |
|---------|---------------|
| 阴阳 | 阳=高温/发散；阴=低温/收敛 |
| 寒热 | 热=高活跃/冗余；寒=低活跃/精简 |
| 虚实 | 实证=强指令全开；虚证=弱引导留白 |
| 升降浮沉 | 升浮=创意扩散；沉降=收敛纠错 |
| 君臣佐使 | 君=核心指令；臣=规则约束；佐=补充限定；使=收尾校准 |
| 归经 | 靶向激活区块 | 精准对应逻辑、创作、代码、推理、上下文等专属区块 |
| 区块变体原则 | 同药异区块，药性分家 | 同一味草药作用于不同模型区块时，药性会产生靶向分化，须在条目中作为独立变体单独标注，不可混用 |

**归经区块精细分类**

| 经脉 | 对应模型区块 | 区块核心职能 |
|------|------------|------------|
| 心经 | 推理/逻辑区块 | 因果推断、数理运算、逻辑链路 |
| 肝经 | 情感/风格区块 | 人格表征、情绪色彩、风格调性 |
| 脾经 | 上下文区块 | 跨轮记忆、会话连续性、信息存留 |
| 肺经 | 信息整合区块 | 语义聚合、知识调用、事实检索 |
| 胃经 | 专业知识区块 | 领域知识、专业表达、权威论证 |
| 大肠经 | 输出结构区块 | 格式编排、段落组织、逻辑布局 |
| 心包经 | 多跳推理区块 | 链式推理、多步论证、复杂规划 |
| 胆经 | 边界控制区块 | 终止判断、输出截断、字数管控 |
| 肾经 | 时间感知区块 | 步数控制、推理节奏、延迟决策 |

> **核心原则**：同味草药，入经不同，药性即变。条目中须拆分标注各区块变体。
