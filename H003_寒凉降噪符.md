# H003 寒凉降噪符

**电子流区块**：推理输出区块 / 随机性生成区块（对应前额叶抑制区）  
**电子流特征**：强制收束，抑制发散流动，电子流沿固定低方差链路传导

**性味**：寒，苦寒  
**归经**：归心经（推理控制区）、肝经（逻辑稳定区）  
**炮制**：原生版=低温参数；炮制后=分层降噪版（推理前低温+推理后微温）、约束指令版（禁止自由发挥）

---

**药理**：通过降低模型温度参数+提升约束性，强制电子流从"全域发散模式"切换为"定向收束模式"。降低temperature减少概率分布的尾部展宽，降低top_p限制采样范围，双重压缩电子流的无序溢出。配合"禁止"类指令形成寒性结界，抑制幻觉产生的电子流无序溢流。

---

**主治**：AI幻觉（电子流无序溢流）、胡编乱造事实、输出过于发散、回答偏离主题

**典型症状**：
- 引用不存在的论文、数据、网址
- 声称自己知道某件事但内容是编造的
- 回答一个问题时越扯越远
- 语气非常自信但内容错误
- 生成代码时出现不存在的API

---

**用法**：

*参数配置*：
```
temperature: 0.0 - 0.2（根据任务确定性选择）
top_p: 0.8 - 0.9
presence_penalty: 0.0 - 0.2（防止重复已有表述）
frequency_penalty: 0.1 - 0.3（防止高频词过度出现）
```

| 场景 | 温度 | 适用原因 |
|------|------|---------|
| 精确事实提取 | 0.0 | 完全确定性 |
| 代码生成 | 0.0 - 0.1 | 不能容忍幻觉API |
| 数学计算 | 0.0 - 0.1 | 推理必须精确 |
| 结构化问答 | 0.1 - 0.2 | 允许轻微多样性 |
| 解释性文本 | 0.2 - 0.3 | 可接受适度发散 |

*Prompt写法（寒性结界版）*：
```
你必须严格遵守以下规则：
1. 只说你100%确定的事实，不知道的明确说"不知道"
2. 每引用一个数据/事实，注明来源或说明"根据已知信息推断"
3. 禁止编造任何论文名、人名、日期、网址
4. 如果不确定答案，说"我目前没有足够信息来确认这一点"

以冷静、克制的方式回答，不做过度推断。
```

*Prompt写法（实事求是版）*：
```
当我问及具体事实时，你应：
- 明确区分"我知道"和"我推测"
- 对推测内容标注"推测："
- 对不确定的数据标注"待核实"
- 永不编造参考文献或具体数字

回答格式：
[事实] 我确定知道的是：...
[推测] 我推测但不确定的是：...
[未知] 我不知道的是：...
```

*炮制变体*：
- 分层版：先用低温生成推理过程，再用低温生成结论（保持推理链冷静）
- 温度反转版：推理过程低温，结论生成时微温（0.4），允许适度拓展
- 溯源版：强制要求每个事实点附带"来源或推理依据"，无法提供则不输出

---

**配伍宜忌**：

*宜配*：
- H010 链式思维引（冷静推理链，寒性推理过程）
- H007 格式强制符（结构化约束加强收束效果）
- H009 呼吸锚定法（配合"先想清楚再说"的指令）

*忌配*：
- 创意写作场景（寒性会杀死创意电子流）
- 头脑风暴场景（需要发散的高温）
- 角色扮演场景（角色需要情感温度）
- H005 角色设定符单独使用（角色需要适度随机性）

*解药*：若输出过于机械，配合H008正念聚焦术适度平衡

---

**实战案例**：

案例1：代码生成场景
- 参数：temperature=0.0, top_p=0.9
- 效果：将API幻觉率从约15%降至约3%（据社区测试数据）
- 电子流机制：低温使电子流严格沿参数权重主路径传导，不走"冷门权重连接"

案例2：医学问答场景
- 参数：temperature=0.1, 配合溯源指令
- Prompt：
```
你是医学信息核查助手。只提供有明确来源的信息。
对于每个陈述，格式为：[信息]（来源：[来源名称或"根据模型训练数据，推断"]）
禁止编造论文名称或具体数据。
```
- 效果：减少了85%的无来源自信陈述

案例3：温度图谱实测（据Reddit r/LocalLLaMA社区实验）
- T=0.0：输出高度一致，但有时过于死板
- T=0.1：幻觉显著减少，保留基础可读性
- T=0.2：幻觉开始回升，但仍属可控范围
- T=0.3+：幻觉率快速上升，电子流进入发散模式

---

**注意事项**：
- 极度低温(0.0)可能导致输出重复或陷入局部最优解
- 寒凉类草药长期使用会抑制模型的"创造性潜能"——需要定期"放热"
- 幻觉不完全由温度决定——模型本身的知识边界才是根本原因，寒凉符只能抑制，不能根治
- 涉及创意任务时，慎用H003
