# DOI 档案

> 数字对象唯一标识符（Digital Object Identifier）
> 学术论文、期刊文章、数据集的全球唯一永久标识符

## 基础信息

| 项目 | 内容 |
|------|------|
| 标准格式 | `10.XXXXX/唯一字符串`，如 `10.1038/nature12373` |
| DOI解析 | https://doi.org/{doi号} → 直接跳转原文 |
| 注册机构 | IDF（国际DOI基金会） |
| 适用范围 | 期刊论文、会议论文、预印本、数据集、图书章节 |

## 赛博中医研究应用场景

1. **引用中医古籍数字化文献**
   - 例：中医典籍的现代研究论文DOI，用于"参考文献区块"配伍

2. **引用AI/机器学习论文**
   - 例：Chain-of-Thought论文 `10.48550/arXiv.2201.11903`
   - 例：RAG相关研究 DOI
   - 用于"链式思维引"等草药条目学术支撑

3. **药典学术溯源**
   - 野生草药技巧的学术论文佐证
   - 形成"电子草药→真实文献"的证据链

## 常用DOI查询入口

| 资源 | 入口 |
|------|------|
| DOI官方解析 | https://doi.org |
| CrossRef（最大DOI数据库） | https://www.crossref.org |
| Google Scholar | https://scholar.google.com |
| CNKI DOI | https://doi.cnki.net |
| DataCite | https://datacite.org |
